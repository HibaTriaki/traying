👋 مرحبًا يا أبطال البرمجة

هل تريد تعلم البرمجة وصناعة الألعاب والتطبيقات؟ هنا ستتعلم لغة بايثون (Python) خطوة بخطوة بطريقة سهلة وممتعة داخل فضاء البرمجة الكبير!

🏆 تحدي اليوم

أنشئ متغيرًا يحمل اسم مدرستك "Code Way" ثم اطبعه على الشاشة.

💡 هل تعلم؟

لغة Python من أكثر لغات البرمجة استعمالًا في العالم وتُستخدم في الذكاء الاصطناعي وصناعة ألعاب الفضاء والمركبات الذكية!

📦 المتغير (Variable)

تخيل أن لديك صندوقًا تضع بداخله ألعابك. المتغير هو صندوق نطلق عليه اسماً ونحفظ فيه المعلومات.

nom = "Code Way"
print(nom)
Code Way

ملاحظة: المتغيرات تحفظ البيانات لكي لا تضيع.

🔢 العدد الصحيح (Integer - int)

العدد الصحيح هو عدد كامل نستخدمه للحساب والعد بدون أي فاصلة عشرية.

age = 10
print(age)
10

🥤 العدد العشري (Float)

العدد العشري هو كل رقم يحتوي على فاصلة ونقطة عشرية لتحديد الأجزاء بدقة.

prix = 12.5
print(prix)
12.5

📝 النص (String)

النص هو أي كلمة أو جملة نكتبها ونضعها بين علامتي تنصيص ليفهمها الكمبيوتر كنص.

ecole = "Code Way"
print(ecole)
Code Way

🚦 القيمة المنطقية (Boolean)

القيمة المنطقية هي اختيار ذكي وحاسم، وتكون إما صحيح True أو خطأ False.

j_aime_python = True
print(j_aime_python)
True

¼ القوائم (Lists)

القائمة تسمح لك بجمع وحفظ عدة عناصر وألعاب مختلفة معاً في مكان واحد مرتب.

fruits = ["Pomme","Banane","Orange"]
print(fruits)
['Pomme', 'Banane', 'Orange']

🤔 الشروط (Conditions)

الذكاء يعني اتخاذ القرار! إذا تحقق شرطك الخاص والمحدد يتم تنفيذ الأمر فوراً.

note = 18

if note >= 10:
    print("Bravo")
Bravo

🔄 الحلقات (Loops)

بدل إعادة كتابة الكود تكراراً، الحلقة الذكية تقوم بتكرار الأوامر بالعدد الذي تريده.

for i in range(5):
    print("Bonjour")
Bonjour x5

⚙️ الدوال (Functions)

الدالة هي مصنع أو مجموعة أوامر نجهزها ونعطيها اسماً لنستعملها عدة مرات بسهولة.

def saluer():
    print("Bonjour Code Way")

saluer()
Bonjour Code Way

📚 ملخص سريع للأبطال

✔ **المتغير** يخزن المعلومات.
✔ **int** للأعداد الصحيحة.
✔ **float** للأعداد العشرية.
✔ **string** للنصوص والكلمات.
✔ **boolean** للقيم المنطقية.
✔ **list** لتخزين عناصر كثيرة.
✔ **conditions** لاتخاذ القرارات الذكية.
✔ **loops** لتكرار الأوامر بسرعة.
✔ **functions** لتنظيم الكود كالمحترفين.
